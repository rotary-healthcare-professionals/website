Uređivanje vijesti

Kratke upute za Karlu, Dragu i druge ovlaštene urednike IRFHP websitea.

Prijava

  1. Otvori app.pagescms.org.
  2. Prijavi se svojim GitHub računom.
  3. Odaberi rotary-healthcare-professionals/website.
  4. U lijevom izborniku odaberi Vijesti.
Karla i Drago moraju imati pristup repozitoriju, a Pages CMS GitHub App mora biti instalirana samo za repo website. Admin sučelje nije dio javnog websitea i ne otvara se preko /admin.

Dodavanje vijesti

  1. Klikni New ili Create.
  2. Upiši naslov, datum i kratak sažetak.
  3. Po želji odaberi naslovnu sliku.
  4. U polje Sadržaj unesi tekst, poveznice i slike.
  5. Pregledaj sadržaj i klikni Save.
Spremanje izrađuje promjenu u GitHub repozitoriju. Nakon uspješnog automatskog builda vijest se objavljuje na websiteu.

Uređivanje ili brisanje

  1. Na popisu vijesti pronađi naslov.
  2. Otvori vijest, napravi izmjene i klikni Save.
  3. Za brisanje otvori vijest, odaberi Delete i potvrdi. GitHub čuva povijest pa se sadržaj može vratiti.

Dobra praksa

  • Naslov neka bude kratak i jasan.
  • Sažetak neka ima jednu do dvije rečenice.
  • Prije uploada smanji velike fotografije; preporučena širina je do 1600 px.
  • Ne unosi lozinke, privatne podatke ili podatke o karticama.
  • Nakon objave otvori website i provjeri vijest.

Ako nešto ne radi

Nemoj ponavljati spremanje mnogo puta. Zabilježi naslov vijesti i poruku greške te se javi developeru ili administratoru GitHub organizacije.
