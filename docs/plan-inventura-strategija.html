Zašto GitHub Pages + Pages CMS
  • Website je potpuno statički, pa GitHub Pages uklanja trošak hostinga i plaćenih deployment sjedala.
  • Pages CMS uređuje sadržaj izravno u GitHub repozitoriju i ne treba vlastiti Node server.
  • Kod, sadržaj i deployment ostaju u GitHub organizaciji udruge, bez ovisnosti o osobnom računu developera.
