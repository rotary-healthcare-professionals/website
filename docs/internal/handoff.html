Trenutačna produkcija i rollback

Stari WordPress na Bluehostu ostaje privremeni fallback sve dok GitHub Pages nije aktivan, provjeren i spreman za DNS cutover.
Javna domena još pokazuje na Bluehost IP 50.87.253.47. Prije DNS promjene zabilježi sve postojeće zapise. Ako cutover ne uspije, vrati prethodne zapise.
