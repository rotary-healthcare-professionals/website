Važne napomene / zamke

DNS nije na Vercelu: rotaryhealthprofessionals.org (+ www) i dalje pokazuje na stari WordPress hosting (50.87.253.47). Nova verzija živi samo na *.vercel.app. Cutover je svjesno odgođen.
  • Astro ostaje 6.x — Keystatic još ne podržava Astro 7. Ne diži major bez provjere Keystatic podrške.
  • Keystatic storage je local (radi samo u npm run dev). Uređivanje uživo dolazi tek u Fazi 2 (github storage).
  • Preview je iza Vercel auth zida — anonimni curl vraća Vercel login. Za dohvat sadržaja koristi Vercel MCP alat web_fetch_vercel_url ili korisnikov browser.
  • Nakon izmjene keystatic.config.ts / astro.config.mjs restartaj dev server (rute se ponovno registriraju).
  • Brand „← Web" u adminu (src/keystatic/Brand.tsx) — logo linka na /. Korisnik treba potvrditi da klik radi; ako Keystatic omota brand u vlastiti link, prebaciti na otvaranje u novom tabu.
  • Slugovi vijesti: iz WP slug-a; ako je prazan/broj, generira se iz naslova (vidi scripts/news-import.mjs).
  • Preview URL-ovi/deploy status: koristi Vercel MCP alate (list_deployments, get_deployment, get_deployment_build_logs). gh CLI je autentikiran.
